# BoxedTitle

export const Template = args => <BoxedTitle {...args} />

`BoxedTitle` displays a text label enclosed in a rectangle. The label is positioned around the edges of a chart, intended as a title.

<ArgsTable of={BoxedTitle} />

`BoxedTitle` is closely related to `BoxedLabel`, but offers a simpler set of props to quickly position labels intended as titles.

## Variants

## Offsets

Boxes can be positioned away from the chart boundary via the prop `offset`.

    <Story

        }}
        component={BoxedTitle}

## Expansion

It is possible to adjust the extent of the box via the prop `expansion`.

<Canvas>
    <Story
        name="small expansion"
        args={{
            variant: 'top',
            boxStyle: { fill: '#ffffff', stroke: '#222222', strokeWidth: '2px' },
            expansion: [0, -2, 0, -2],
            children: 'Title',
        }}
        component={BoxedTitle}
        decorators={[ChartDecorator]}
    >
        {Template.bind({})}
    </Story>
    <Story
        name="large expansion"
        args={{
            variant: 'top',
            boxStyle: { fill: '#ffffff', stroke: '#222222', strokeWidth: '2px' },
            expansion: [0, 30, 0, 30],
            offset: 0,
            children: 'Title',
        }}
        component={BoxedTitle}
        decorators={[ChartDecorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>

Note a small negative expansion will shrink the box, and can balance a non-zero color border.

## Content

The content of the label can be an object
