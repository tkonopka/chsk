# FilterInsetShadow

export const Template = args => <FilterInsetShadow {...args} />

`FilterInsetShadow` creates an svg filter that creates a lighter-color mask.

<ArgsTable of={FilterInsetShadow} />

## Blur

Prop `blurStdDeviation` determines the amount of blur for the shadow.
Larger values makes the shadow spread further into the target object.
