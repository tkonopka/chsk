# BandLabels

<Meta title="Addons/Band/Bands/BandLabels" />
export const Template = args => <BandLabels {...args} />

`BandLabels` draws a set of labels on a bar chart.

<ArgsTable of={BandLabels} />

`BandLabels` is similar to `BarsLabels`. However, whereas `BarsLabels` draws labels associated with bars (i.e. rectangles), this component draws labels associated with bands (i.e. groups of bars labeled on the index axis).

## Position & content

Band labels are typically displayed at the end of value axis, setting prop `position` to 1.

The content of the label is determined by a function set with prop `format`.
This can be used, for example, to compute the sum of values in the band.
