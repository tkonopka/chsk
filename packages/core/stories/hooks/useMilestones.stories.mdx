# `useMilestones`

The `useMilestones` hook retrieves a set of milestone strings.

<Source
    language="javascript"
    code={`
import { useMilestones } from '@chsk/core'

// in a JSX component:
const milestones = useMilestones()
`} />

Recall milestone strings are used to trigger animations with `MilestoneMotion`.
