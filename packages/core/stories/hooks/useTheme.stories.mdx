# `useTheme`

The `useTheme` hook returns an object with all settings encoded in the current theme.

<Source
    language="javascript"
    code={`
import { useTheme } from '@chask/core'

const theme = useTheme()
`} />

The resulting theme object contains merged information from the default theme and from any custom settings provided to the `Chart` component.
