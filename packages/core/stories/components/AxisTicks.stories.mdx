# AxisTicks

<Meta title="Core/Components/AxisTicks" />
export const Template = args => <AxisTicks {...args} />

`AxisTicks` draws a set of tick line and labels. `AxisTicks` should be used within an `Axis` component.

<ArgsTable of={AxisTicks} />

## Variants

Predefined variants draw ticks on the top, right, bottom, or left axis. For brevity, the example below shows ticks on the bottom axis.

<Canvas>
    <Story
        name="default"
        args={{
            variant: 'bottom',
        }}
        component={AxisTicks}
        decorators={[ChartBottomAxisDecorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>

## Tick marks

The position of ticks can be specified as an integer (to suggest a number of tick marks) or as an array (to specify all the tick positions).

<Canvas>
    <Story
        name="number"
        args={{
            variant: 'bottom',
            ticks: 10,
        }}
        component={AxisTicks}
        decorators={[ChartBottomAxisDecorator]}
    >
        {Template.bind({})}
    </Story>
    <Story
        name="positions"
        args={{
            variant: 'bottom',
            ticks: [0, 20, 50, 90, 100],
        }}
        component={AxisTicks}
        decorators={[ChartBottomAxisDecorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>

Note that suggestions for the number of tick marks are approximate and the actual number of ticks may differ; the actual positions are computed by [d3-scale](https://github.com/d3/d3-scale).

## Label position

The position of the text labels near tick marks is controlled with props `labelOffset` and `labelTranslate`.

`labelOffset` determines a perpendicular distance from the axis to the text label.
`labelTranslate` further adjusts the position in an arbitrary direction.

<Canvas>
    <Story
        name="label offset"
        args={{
            variant: 'bottom',
            labelOffset: 25,
        }}
        component={AxisTicks}
        decorators={[ChartBottomAxisDecorator]}
    >
        {Template.bind({})}
    </Story>
    <Story
        name="label translate"
        args={{
            variant: 'bottom',
            labelOffset: 5,
            labelTranslate: [10, 5],
        }}
        component={AxisTicks}
        decorators={[ChartBottomAxisDecorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>

## Rotation

Tick labels can be rotated using the prop `labelRotate`. Note that adjusting the angle will likely prompt adjustment to css properties like `textAnchor` and `dominantBaseline`.

<Canvas>
    <Story
        name="rotation"
        args={{
            variant: 'bottom',
            labelOffset: 10,
            labelRotate: -90,
            labelStyle: {
                textAnchor: 'end',
                dominantBaseline: 'middle',
            },
        }}
        component={AxisTicks}
        decorators={[ChartBottomAxisDecorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>

## Label format

Labels can be formatted with a custom function through the prop `labelFormat`.

        component={AxisTicks}
        decorators={[ChartBottomAxisDecorator]}
    >
        {Template.bind({})}

</Canvas>
